Welcome to Vidrovr Python SDK's documentation!
==============================================

Vidrovr's Python SDK is a Python library for accessing the Vidrovr platform which makes all parts of video
useful to empower enterprise decision makers and alleviate tedious work.

Check out the :doc:`usage` section for further information, including how to
:ref:`install <installation>` the project.

.. note::
   This project is under active development.
