Settings#

Updating settings data#

To update language settings for a specific project, you can use the vidrovr.resources.settings.Language object.

pydantic model src.vidrovr.resources.settings.LanguageModel#

Model of language settings.

Parameters

  • audio_input (str) – Spoken language of the source in the format of ISO langauge code-ISO Region code (en-US)

  • on_screen_input (str) – Language of the text written on screen of the source in the format of ISO language code (en)

  • output (str) – Language of the transcript produced by the platform in the format of ISO language code (en)
