Custom Tags#

Creating, deleting and reading tag data#

To create a new tag in a project, delete a tag from a project or retrieve a list of tags for a given project or the details of a specific tag, you can use the vidrovr.resources.CustomTag object along with the CustomTagModel data object:

pydantic model src.vidrovr.resources.custom_tags.CustomTagModel#

Model of a custom tag.

Parameters

  • id (str) – ID of the custom tag

  • name (str) – Name of the custom tag

  • type (str) – Type of tag

  • is_active (bool) – Indicates if the custom tag is active or not

  • creation_date (str) – Date the custom tag was created

  • num_examples (int) – Number of examples for the custom tag

  • num_reviewed (int) – Number of reviewd examples for the custom tag

  • num_unreviewed (int) – Number of unreviewed examples for the custom tag

Creating, reading and updating tag examples#

To create a new tag example in a project, update the example or retrieve a list of tag examples in the project, you can use the vidrovr.resources.CustomTagExample along with the CustomTagExampleModel data object:

pydantic model src.vidrovr.resources.custom_tags.CustomTagExampleModel#

Model of a custom tag example.

Parameters

  • id (str) – ID of the custom tag example

  • href (str) – Cloudfront URL of the tag example image

  • source (str) – Source of the tag example - search or user_upload

  • status (str) – Status of the tag example - accepted, rejected or candidate

src.vidrovr.resources.custom_tags.CustomTagExample.read(project_id: str, tag_id: str, example_id: Optional[str] = None)#

Returns either a single tag example or an array of tag examples identifiers, paginated, and filtered by status, which can be one of three values: accepted, rejected or candidate. The first value of page is always 0.

Parameters

  • project_id (str) – ID of the project from which to retrieve tag examples

  • tag_id (str) – ID of the tag from which to retrieve examples

  • example_id (str) – ID of the example to retrieve or None

Returns

An array of tag examples or a single example

Return type

list[CustomTagExampleModel] or CustomTagExampleModel
